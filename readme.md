# pinout
![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/rainbow.png)


## buttons

 button0 = pin 0
 button1 = pin 1
 button2 = pin 2
 button3 = pin 3
 button4 = pin 4
 button5 = pin 5
 button7 = pin 10
 button8 = pin 11

 ## audio shield (automatic but good to remember)
 MCLK - 23
 BCLK - 21
 LRCLK - 20
DIN (audio output) - 7
